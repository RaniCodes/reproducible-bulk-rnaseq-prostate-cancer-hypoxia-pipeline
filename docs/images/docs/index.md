# Reproducible bulk RNA‑seq pipeline for prostate cancer hypoxia

This site documents an end‑to‑end bulk RNA‑seq workflow for LNCaP and PC3
prostate cancer cell lines (hypoxia vs normoxia), implemented on an
8 GB RAM WSL laptop.
